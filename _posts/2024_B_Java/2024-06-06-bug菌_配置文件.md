## <a id="content1">配置文件介绍</a>

配置文件有两种    

**application.properties**    
```text
server.port = 8060
spring.profiles.active = dev
HelloModel.userName = 刘亦菲
HelloModel.sex = 女
HelloModel.age = 20
```

**application.yaml**    
```text
server:
  port: 8060
  
spring:
    profiles:
        active: dev
    
HelloModel:    
  username: 刘亦菲
  sex: 女
  age: 20
```



## <a id="content2">配置文件优先级</a>     

在 Spring Boot 中，一共有4个地方可以存放该配置文件，并按照如下优先级依次降低，如下:        
1.当前项目根目录的config下    
2.当前项目根目录下    
3.resource目录的config目录下    
4.resource目录下     


## <a id="content3">属性注入</a>

通过属性注入的方式使用配置文件的数据

```text
@Component
public class User {
 
    @Value("${user.userName}")
    private String userName;
    @Value("${user.sex}")
    private String sex; 
    @Value("${user.age}")
    private Integer age;
 
public String getUserName() {
    return userName;
}
public void setUserName(String userName) {
    this.userName = userName;
}    public String getSex() {
        return sex;
    }
    public void setSex(String sex) {
        this.sex = sex;
    }
    public Integer getAge() {
        return age; 
   }    public void setAge(Integer age) {
        this.age = age;
   }
}
```
