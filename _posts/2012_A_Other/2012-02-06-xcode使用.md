## <a id="content1">快捷键</a>

```
command + option + [ 向上交换代码
command + option + ] 向下交换代码
command + \  打开关闭断点
command + L 定位到具体某一行


control + k 删除光标右边一行的内容
control + d 删除光标右边一个字符
control + 6 定位到某个方法
control + L 光标在窗口居中
```
