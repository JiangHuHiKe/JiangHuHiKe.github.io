## <a id="content1"></a>查看所属内存段

```objc
int age = 18;
int main(int argc, const char * argv[]) {
    @autoreleasepool {
        int a = 10;
        NSLog(@"数据区：age    %p",&age);
        NSLog(@"栈区：  a      %p",&a);
        NSLog(@"堆区：  Obj    %p",[[NSObject alloc]init]);
        NSLog(@"数据区：Person %p",[Person class]);
    }
    return 0;
}
```

打印日志
```objc
数据区：age    0x1000011d0
栈区：  a      0x7ffeefbff5cc
堆区：  Obj    0x100700100
数据区：Person 0x1000011a8
```
