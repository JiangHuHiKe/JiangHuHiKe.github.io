## <a id="content1">01 高手的暗箱</a>

每个时代的高手都在利用<span style="color:red;font-weight:bold;">社会和科技的底层逻辑</span>撬动自己，实现跨越式成长。     
<span style="color:grey;font-size:12px;">
1、社会的底层逻辑，就是那些不变的规律。社会是人组成的，人性构成了社会运行的本质。    
2、要顺应时代的发展。     
3、科技就是自己的左膀右臂，提高自己的效率。        
</span>


#### **走在时代前面的明白人**     

认知方式的改变：调用知识而非记忆知识     
<span style="color:grey;font-size:12px;">
1、在现在社会知道去哪里获取知识，知道知识的原理，比单纯的背诵并记住知识更重要。                        
</span>

思考方式的改变：从独自思考到联机的独立思考        
<span style="color:grey;font-size:12px;">
1、思想需要碰撞才更能接近真理。从网络接受不同的思想，跟别人沟通获取不同的思想。    
2、实践 - 吸收并总结归纳，再深入思考 - 实践。      
</span>

核心竞争力的改变：人机合一     
一个人能够<span style="color:red;">用机器学习和处理信息，用大脑整合和创新思想，用系统思维思考问题</span>，会是未来最有竞争力的    
<span style="color:red;font-weight:bold;">AI时代，究竟人的什么能力是最重要不可被替代的？？？<br>如何获得这种能力？？？</span>     
<span style="color:grey;font-size:12px;">
我今天做的事情，机器能做吗？    
我今天做的事情，会被外包吗？       
我今天做的事情，明天会做的更好吗？         
</span>


#### **拉斐尔也用投影仪**    
各个领域的高手们站在了哪些看不见的巨人之上？他们有哪些“暗箱”？     

#### **个体的跨越式成长**    
只要你升级心智、洞察趋势、聪明地勤奋，人人都是享受时代红利的幸运儿。     
<span style="color:grey;font-size:12px;">
1、心智升级：汲取能量-思考-实践，提升自己的认知。                 
2、洞察趋势：站在风口上，猪也能飞起来。        
3、聪明地勤奋：战略上的勤奋，不是战术上的勤奋。   
就是不要埋头苦干，重复劳动。要抬头看路，思考方向。      
</span>

<span style="color:grey;font-size:12px;">

## <a id="content2">02 高手战略</a>

竞争越是开放，个人越需要打磨深思熟虑后做选择的战略能力--找到那些“更少但是更好”的事。       
<span style="color:red;">决策和选择的能力很重要</span>    

你投入练习，产出技能；投入享乐，产出体验； 投入情感，产出关系； 投入学习，产出认知。     
每个人都是自己的投资人——早上拿到24小时的时间精力，晚上账户结算，第二天早上重新开始。    
决定一个人几年后不同的，正是那些你睡 着以后，能够持续迭代的东西。       

#### **只打甜蜜区里的球**     

泰德在其影响深远的教科书《击打的科学》 （The Science of Hitting）中，提出一个观点：高击打率的秘诀是不要每个球都打，只打“甜蜜区”的球。正确地击打甜蜜区的球，忽略其他 区域的，就能保持最好成绩。    

股神巴菲特的投资理念：只投资高价值、可迭代、有护城河的公司，其他的不看；不求短期获利，只看长期获利，尽量少动。

大蛇战略：善猎者善于等待

#### **新东方名师的崛起路径**   

非常努力地观察、发现和验证，最终才能逐渐通过乱象，发现高价值区。     

高价值区的特点    
第一，高价值区往往是反直觉、说了你都不信的区域。   
第二，高价值区需要高竞争力。    
第三，高价值区随着时代在改变。    

新东方名师，通过总结出好教师的5项技能，形成了自己的教学方法论，这是新东方名师的甜蜜区。     
<span style="color:gray;font-size:12px">5项技能通过快速循环型掌握每个技能的80%，而不是死磕1项到100分</span>     

学而思通过小班战略，培优而不是补差，这是学而思的甜蜜区。       


#### **幂律分布：发现世界的杠杆点**    

马太效应、长尾分布、二八定律。     

正态分布 向 长尾分布 的变迁，是趋势。所以阶层分化是开放社会的必然趋势。      

<span style="color:red;">停止抱怨。世界是不公平的，接受它。</span>        
<span style="color:red;">持续学习 = 扩大视野。提高效能 = 扩大移动能力。</span>      
<span style="color:red;">持续关注、观察、验证高价值区。</span>               
<span style="color:red;">向正确的方向移动，爬上幂律顶部。</span>     

#### **头部效应：站位比努力更重要**     

**1️⃣ 这是一个“头部拿走大多数回报”的时代**    
资源与机会呈现极端不均匀分布，少数头部个体占据了大部分注意力、资源与回报，平均水平正在迅速贬值。

**2️⃣ 努力本身不再决定结果，位置决定努力的价值**    
同样的努力，放在头部位置会被放大，放在边缘位置会被稀释甚至浪费；选择在哪里努力，比努力本身更重要。      
<span style="color:red;">在正确的方向和正确的位置努力，就是这个位置有没有“头部效应”收益巨大。</span>     
<span style="color:red;">努力能不能被“重复利用”。真正会被放大的位置，有一个特征： 今天的努力会在明天、下一个项目、下一个阶段继续生效</span>     


**3️⃣ 站位的本质是“被清晰识别和优先选择”**     
头部不是最努力的人，而是最容易被记住、被找到、被选择的人；模糊的能力等同于没有位置。     
<span style="color:red;">某一个领域或细分方向的专业人士，提到这个领域就会想到你。要能被看见，然后被选择。</span>     


**4️⃣ 真正有效的成长路径，是先进入“小范围头部”**      
个人跃迁并非一步登顶，而是通过缩小战场、进入局部头部，再借助正反馈扩展到更大的舞台。

**5️⃣ 中间层是最危险的状态**   
“还不错但不突出”的平均层，既没有淘汰压力，也得不到放大效应，是最消耗努力、最难跃迁的位置。


#### **用头部效应就业、择城、选创业赛道**     

**用头部效应就业**    
核心观点： 在职业发展中，你的“站位”比努力更重要。   
头部公司效应：加入行业顶尖公司，能让你的能力和影响力被放大。         
同样努力，不同位置，结果不同：在小公司做同样努力，可能没人看见；在头部公司做，别人会注意你。     
建议：        
- 初期尽量进入头部企业或团队，快速积累经验和人脉。       
- <span style="color:red;">注意自己的“可见度”和“影响力”</span>，不仅是完成工作。       

**选对城市，多资源**      
资源集中的城市，成长更快，机会更多。      

**挑对赛道，事半功倍**     
创业要选有潜力的赛道，站在风口上才能放大价值。


#### **专注：高手的护城河 ⭐** 

高手的领先，不是靠天赋或努力，而是靠把<span style="color:red;">精力集中在关键领域</span>，形成别人难以复制的深度能力。     
<span style="color:red;">长期专注会产生复利效应，让经验、认知和判断力不断积累，形成天然护城河。</span>         
<span style="color:red;">专注意味着选择性放弃</span>，把精力投向最有价值的方向，从而获得认知优势和执行优势。    


#### **迭代：聪明人的笨功夫 ⭐**    

**迭代本质**             
通过反复尝试和改进，把笨功夫转化为高效方法。       
妙手是成为高手的最大障碍，<span style="color:red;">迭代的手艺</span>才是征途。        

**小步快跑**           
快速迭代，及时发现问题，形成反馈循环。

**聪明人的技巧**    
用规律和工具降低重复成本，实现质的飞跃。   

在当今社会，只要你愿意用时间打磨一门手艺，就会有自己的护城河。     


## <a id="content3">03 联机学习</a>

在知识爆炸、终身学习时代，人与人之间比拼的不是学与不学，而是<span style="color:red;font-weight:bold;">认知效率</span>。        
学习前，想明白<span style="color:red;font-weight:bold;">学什么、怎么学、有什么用和如何兑现。</span>

#### **功利学习法：学得更好，却学得更少 ⭐**      

<span style="color:blue;font-weight:bold;">识别知识的源头</span>     
一手信息：知识的源头       
二手信息：忠实转述一手信息         
三手信息：为传播而简化和极端化观点的陈述           
四手信息：出于各种动机充满个人经验的情绪化表达     
所以在知识爆炸的年代，最好的方式是:     
<span style="color:red;">辨别一二三四手信息，走向知识的源头，并与那些人站在一起。总有一天，你也会成为创造知识的人。</span>


<span style="color:blue;font-weight:bold;">功利读书法</span>      
什么是「功利学习法」？          
功利学习法不是急功近利，而是结果导向。     
它强调： 只学习“能改变你现实结果”的那一小部分知识。     

为什么要「学得更少」？   
知识无限，但个人精力极其有限     
<span style="color:red;">学习的真正成本是时间 + 注意力</span>         
<span style="color:red;">80% 的结果，来自 20% 的关键知识</span>            

功利学习法的核心原则    
先确定目标，再倒推学习内容，为了达成这个目标，我必须掌握哪几个关键点？    
只学「最短路径」上的知识，能解决问题 → 学，只是拓展认知 → 暂不学。     
学习必须立刻进入使用场景，不会用的知识，本质上等于没学。          

功利学习法的本质   
<span style="color:red;">用“现实反馈”筛选知识，用“结果变化”衡量学习质量。</span>    
<span style="color:red;">高手不是学得最多的人，而是最清楚“什么不该学”的人。</span>    


<span style="color:blue;font-weight:bold;">萃取知识晶体</span>       
如何更好的管理自己学到的知识，当需要的时候能够输出。     
4中常见的知识结构：关联、树状、序列、数据。

<span style="color:red;font-weight:bold;">找到知识源头、极其功利地读书、萃取知识晶体，这就是提高认知效率的核心方式。</span>
