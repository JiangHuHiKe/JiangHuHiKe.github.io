## <a id="content1">man命令</a>

- [参考文章：man命令常用工具命令](https://man.linuxde.net/man)

man命令是Linux下的帮助指令，通过man指令可以查看Linux中的指令帮助、配置文件帮助和编程帮助等信息。

比如查看 git 指令,在终端输入

```
man git
```

搜索git相关指令说明

```
#比如查找 push 的说明
/-push

/ : 开始查找
n : 查找下一个
N : 查找上一个
q : 退出
```
