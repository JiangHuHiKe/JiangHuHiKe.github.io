## <a id="content1"></a> 283. 移动零

**题目描述**

```
 283. 移动零
 
 给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

 示例:
 输入: [0,1,0,3,12]
 输出: [1,3,12,0,0]
 
 说明:
 必须在原数组上操作，不能拷贝额外的数组。
 尽量减少操作次数

 来源：力扣（LeetCode）
 链接：https://leetcode-cn.com/problems/move-zeroes
 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```


**题目实现 - 方式一**

```
class Solution1 {
public:
    void moveZeroes(vector<int>& nums) {
        for (int i = 0,cur = 0; i<nums.size(); i++) {
            
            if (nums[cur] !=0 && nums[i] != 0) {
                //同时为0，cur++
                cur++;
            }else if(nums[cur] == 0 && nums[i] != 0){
                //该情况交换
                nums[cur] = nums[i];
                nums[i] = 0;
                cur++;
            }
        }
    }
};
```

**题目实现 - 方式二**

```
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        for (int i = 0,cur = 0; i<nums.size(); i++) {
            
            //如果是0的话就跳过
            if (nums[i] == 0) continue;
            
            //cur != i 证明cur在为0的位置停住了
            //num[cur]==0;num[i]!=0
            if (cur != i) {
                nums[cur] = nums[i];
                nums[i] = 0;
            }
            
            cur++;
        }
    }
};
```

## <a id="content2"></a> 1. 两数之和


**题目描述**

```
1. 两数之和
给定一个整数数组 nums 和一个目标值 target，
 请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。
 你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。

 示例:
 给定 nums = [2, 7, 11, 15], target = 9
 因为 nums[0] + nums[1] = 2 + 7 = 9
 所以返回 [0, 1]

 来源：力扣（LeetCode）
 链接：https://leetcode-cn.com/problems/two-sum
 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
```

**题目实现 - 方式一**

```
//暴力法
//暴力法
class Solution1 {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector<int> result(0);
        int size = (int)nums.size();
        if (size == 0) return result;
       
        for (int i = 0; i<size;i++) {
            for (int j = i+1; j<size; j++) {
                if (nums[i] + nums[j] == target) {
                    result.push_back(i);
                    result.push_back(j);
                }
            }
        }
        return result;
    }
};
```


**题目实现 - 方式二**

```
//利用map
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector<int> result(0);
        int size = (int)nums.size();
        if (size == 0) return result;
        
        map<int, int> numIdx;
       
        for (int i = 0; i<size;i++) {
           
            //查看map内是否有需要的值
            int pre = target - nums[i];
            if (numIdx.count(pre)>0) {
                result.push_back(numIdx[pre]);
                result.push_back(i);
            }
            
            //存入map,num:idx
            numIdx[nums[i]] = i;
        }
        
        return result;
    }
};
```
